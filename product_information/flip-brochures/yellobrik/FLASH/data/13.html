PDM 1284 B Application

PDM 1284 B modules can be cascaded in this way to add more audio 

channels. 2 x PDM 1284 B modules can be cascaded to transport 16 audio 

channels over a single fiber link. Each module would have different AES 

groups selected (GRP 1,2,3,4). In this example “auto-black” mode is selected, 

so no SDI video input is required.
