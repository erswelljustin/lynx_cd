RFR 1001

Module Mounting Bracket for Single Yellobrik

• Robust metal mounting bracket
• Mount on any flat surface
• Ideal for mounting on 19” rack rails
• No tools needed for module installation

 

The RFR 1001 is a robust metal mounting solution for a single 

yellobrik. The bracket can accommodate the smaller and larger 

modules using the mounting slots provided in the yellobrik.

The bracket can be mounted on any flat surface using suitable 

screws or bolts (not supplied). The mounting holes are on 19” rack 

rail centers which makes it ideal for mounting yellobriks in the 

rear of equipment rack frames; keeping them secure and out of 

the way.

No tools are required for module installation and removal, this is 

accomplished using a nylon thumbscrew.
