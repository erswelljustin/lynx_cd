RedundantControl

The OH_RCT5023_SERVER provides great additional features that greatly increase the functionalities of the APPolo Control System. Of course operating safety needs to be ensured even if the server should fail. This is what the RedundantControl Server feature provides.

  • The redundant servers will synchronize each other automatically
  • Automatic failover
  • The server with the redundant server option activated will loan all options of the server it is configured to be redundant to

  • Available as software option (OC_REDUNDANT_SERVER).
    Can be activated at any time.
