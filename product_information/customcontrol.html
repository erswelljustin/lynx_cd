CustomControl

This feature is an option for the server enabling the possibility to design virtual control panels and assigning these to individual users.

  • Any parameter of all the modules within the APPolo control system can be controlled or monitored via the virtual control panels
  • Design a comprehensive panel with the zoom in and out feature of the felxGUI
  • The virtual control panels can be assigned to one or more users.
  • Access the panels either from a PC or with an iPad
  • Add virtual control panels to the classical APPolo GUI main control view
  • Easy to use panel configurator

  • Available as software option (OC_CUSTOM_CONTROL) for the OH_RCT5023_SERVER.
    Can be activated at any time.
